# Agents

A collection of AI agents built with Claude.

## Available Agents

| Agent | Description | Stack |
|-------|-------------|-------|
| [street_art_agent](./street_art_agent) | Researches a city's street art scene and generates advertising campaign concepts with AI-generated imagery | React, Claude Sonnet, Stable Horde |

---

Each agent ships with a `README.md`, `manifest.json`, `.env.example`, and `setup.sh` for one-command installation.
